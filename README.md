# The Equivalence T-Test Module

## Overview

<img src='inst/icons/equivalence-module.svg' width='149' height='173' align='right'/>

**Equivalence T-Test module** is an add-on module for JASP that implements both classical and Bayesian equivalence t-tests. The module supports one-sample, paired, and independent-samples designs.
